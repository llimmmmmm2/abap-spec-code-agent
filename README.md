# abap-spec-code-agent

spece 부터 code 생성까지 하는 abap 에이전트



\# ABAP Spec/Code 생성 에이전트\_TEST



SAP ABAP 개발 과정에서 요구사항을 구조화하여 SPEC(설계서)와 CODE 초안을 생성하는 AI 기반 개발 보조 에이전트입니다.



이 프로그램은 요구사항 입력 → SPEC 생성 → SPEC 확정 → CODE 생성 과정을 단계적으로 지원하며, 생성 결과와 사용자 로그를 기록하여 향후 규칙 개선에 활용할 수 있도록 설계되었습니다.



\---



\# 대상 사용자



본 프로그램은 완전한 비개발자를 위한 도구가 아니라 다음과 같은 사용자를 대상으로 합니다.



\- SAP 운영 인력

\- SAP 업무 담당자

\- ABAP 개발 경험이 많지 않은 사용자

\- SAP 테이블 / 구조 / 필드 개념을 이해하고 있는 사용자

\# 개발 배경



최근 AI 기술이 개발 업무에 도입되면서 단순 코드 작성 작업은 점차 자동화될 가능성이 높아지고 있습니다.



이에 따라 운영 인력 역시 단순히 코드를 작성하는 역할보다는 요구사항을 정리하고 설계를 구조화하는 업무에 더 집중할 필요가 있다고 판단하였습니다.



작년에는 프롬ABAP을 활용하여 ABAP 코드 생성 가능성을 테스트하였으나, 코드 생성 이전 단계인 SPEC 작성 과정이 자동화되지 않아 요구사항을 직접 정리하여 단계적으로 입력해야 하는 번거로움이 있었습니다.



이러한 경험을 바탕으로 요구사항을 코드 구현이 가능한 수준의 SPEC으로 구조화하는 에이전트를 구현하게 되었습니다.

\# 화면 안내



\## 1. 요구사항 입력



사용자가 프로그램 요구사항을 다음 항목으로 나누어 입력합니다.



\- 기본 정보

\- 업무 개요

\- 데이터 정의

\- 화면 구성

\- 상태 / 권한

\- 범위 정의



\---



\## 2. SPEC 생성



입력된 요구사항을 기반으로 SPEC 초안을 생성합니다.



\---



\## 3. 미확정 항목 보완



SPEC 초안에서 확정되지 않은 항목을 사용자 입력으로 보완합니다.



\---



\## 4. 최종 SPEC 생성



보완된 정보를 기반으로 최종 SPEC을 생성하고 다운로드할 수 있습니다.



\---



\## 5. CODE 생성



확정된 SPEC을 기반으로 ABAP CODE 초안을 생성합니다.
